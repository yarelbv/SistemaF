¿Cuál es el propósito de los métodos de acceso ?

Crea un programa en Python que simule la lectura y escritura de datos desde y hacia un archivo utilizando métodos de acceso. El programa debe tener las siguientes funciones:

  • escribir_en_archivo(datos, nombre_archivo): Esta función toma una lista de datos y un nombre de archivo como parámetros y escribe esos datos en el archivo. Cada elemento de la lista debe escribirse en una nueva línea.
  • leer_desde_archivo(nombre_archivo): Esta función lee los datos del archivo especificado y los muestra en la consola.
