Variables

Una variable, es conceptualmente un recipiente donde guardamos un dato. Las variables se definen siempre con un tipo asociado. No se puede meter en una variable ningún dato que no sea del tipo con que se definió.

Una variable es donde se almacenan y se recuperan los datos de un programa. En programación, la utilizamos para guardar datos y estados, asignar ciertos valores de variables a otras, representar valores de expresiones matemáticas y mostrar valores por pantallas. Las variables se expresan así:

  • Como un número
  • Como un texto
  • Como un dato abstracto
  • Como un objeto
