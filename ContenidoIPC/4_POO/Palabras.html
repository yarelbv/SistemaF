Palabras reservadas

Las palabras reservadas o palabras clave, tienen un significado especial para el compilador de cualquier lenguaje de programación. Estas palabras pueden identificar los tipos de datos que se pueden usar, además de las diferentes rutinas de programación que permite cada lenguaje. En lenguajes de programación orientado a objetos, puedes ver estas palabras clave:

  • public
  • protected
  • private
  • class
  • abstract
  • interface
  • this
  • friend
